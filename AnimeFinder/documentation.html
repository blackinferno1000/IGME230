Explanation

Anime Finder is a tool to search for and discover anime. It's a web service using the Kitsu API to ajax for data. The design includes a searchbar limit and genre pickers and buttons for functionality, navigation, and finding what's popular. Webstorage is used in the form of a history log. This can be toggled to be seen or not so as to not take up too much space.
